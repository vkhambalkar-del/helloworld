[[data-models]]
== Data Models

[[model-helloworld]]
=== HelloWorld

The HelloWorld object represents a greeting response.

[cols="1,1,1,3"]
|===
|Field |Type |Required |Description

|`message`
|String
|Yes
|The greeting message. Must be between 1 and 100 characters.
|===

==== Example

[source,json]
----
{
  "message": "Hello, World!"
}
----
